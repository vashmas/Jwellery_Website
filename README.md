# VS Jwellers 💍

A full-stack e-commerce jewelry website built with React, PHP, and MySQL.

## 🛠️ Tech Stack
- **Frontend:** React, React Router, CSS
- **Backend:** PHP, REST API
- **Database:** MySQL
- **Tools:** XAMPP, phpMyAdmin

## ✨ Features
- Beautiful Black & Gold luxury design
- Product catalog with category filter
- Product detail page with description
- WhatsApp order integration
- Admin panel with login
- Product add/delete with image upload
- Fully connected to MySQL database

## 📁 Pages
- Home — Featured products
- Products — Full catalog with filters
- Product Detail — Single product view
- About — Brand story
- Contact — Social media & WhatsApp

## 👤 Admin Panel
- Secure login
- Add/delete products
- Image upload
